# Lattice ML Platform

> A production-oriented machine learning platform for predicting the mechanical properties of additively manufactured lattice structures.

## Overview

Lattice ML Platform is an ongoing project that transforms my PhD research into a production-ready machine learning application following modern MLOps practices.

The project focuses on predicting the **Young's Modulus** of nested lattice structures fabricated using Vat Photopolymerization (VPP) additive manufacturing. Rather than stopping at model development, this repository demonstrates how research-grade machine learning models can be engineered, automated, containerized, tested, and deployed using industry-standard tools.

This repository is being developed incrementally as I learn and apply software engineering and MLOps best practices.

---

## Project Objectives

- Build reproducible machine learning pipelines
- Automate data preprocessing and model training
- Compare multiple regression algorithms
- Track experiments and model versions
- Package models using Docker
- Develop REST APIs for inference using FastAPI
- Implement CI/CD using GitHub Actions
- Deploy the application to the cloud
- Follow professional software engineering practices

## Research Background

This project is based on my PhD research in additive manufacturing, where machine learning models are developed to predict the Young's Modulus of nested lattice structures using finite element simulation data.

The long-term goal is to bridge the gap between academic machine learning research and production-ready engineering systems.
